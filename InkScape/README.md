# Parameter für InkScape

Alle Angaben sind in "mm"



##  Scala: Extension "Knob Scale"

Nach dem Hinzufügen die Linienstärke auf 0.3mm ändern!


### 12 Stunden
* Allgemein
  * Center X: 29
  * Center Y: 32
  * KnobSize: 15
  * Angle: 90
  * Linienstärke: 1
  * Draw Arc
* Markierungen
  * Number of tick marks: 12
  * Tick Size: 3
  * Number of subticks: 0
  * Subtick size: 1
  * Scale style: Marks outwards
* Beschriftungen
  * Enable Labels
  * Rounding Float to 0
  * Labels Size: 2
  * Versatz: 2
  * Start Value: 1
  * Stop Value: 12
